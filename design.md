# Design Spec for Todo App

## 1. Screens
- Home Screen: Список всех задач.

## 2. Data (Task Model)
- id: уникальный номер
- title: текст задачи
- isCompleted: булево значение (выполнено или нет)

## 3. Actions
- Add Task: Добавить новую задачу в список.
- Toggle Status: Нажать на чекбокс, чтобы изменить статус.
- Delete Task: Удалить задачу из списка.
